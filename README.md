# js-weather
A simple weather app to get familiar with asynchronous JavaScript

![screenshot](./screenshot.png)

## Installation

```
npm install
npm run build
```

## Run the server

```
npm run start
```

## Live version

A live version is available [here](https://raw.githack.com/elshaka/js-weather/weather-app/dist/index.html)
